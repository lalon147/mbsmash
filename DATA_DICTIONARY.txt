REGISTRATION
  Type: Text (Vehicle registration number)
  Example: 1XS3FS, U2765, R2855
  Purpose: Groups parts by vehicle
  You Update: NO (already in your data)
  Notes: Some cells might be blank if part wasn't assigned to a vehicle

STATUS
  Type: Choice field (should be one of: Ordered, Received, Return, Cancelled)
  Current Values: Mixed (Received, RECEIVED, ordered, etc.)
  Purpose: Track where each part is in its lifecycle
  You Update: YES - Change this when parts arrive
  Example: Change from "Ordered" → "Received" when it shows up

DAYS WAITING (AUTO-CALCULATED)
  Type: Number (days)
  Formula: =IF(AND(F{row}<>"",H{row}=""), TODAY()-F{row}, "")
  Meaning: TODAY() - ORDER_DATE, but only if not yet received
  Purpose: Shows how long you've been waiting
  You Update: NO - System calculates automatically
  
  Example:
  - Order Date: 2025-08-21
  - Today: 2025-08-30
  - Days Waiting: 9 days
  
  When to Worry:
  - >7 days = Follow up with supplier
  - >14 days = More urgent follow-up
  - >30 days = Flag for return/replacement

DASHBOARD SHEET - METRICS EXPLAINED
================================================================================

TOTAL PARTS ORDERED
  Formula: =COUNTA('Master Inventory'!C2:C960)
  What it means: Count of all rows with part names
  Should equal: 959 (your data)
  Why: Quick snapshot of workload

PARTS RECEIVED
  Formula: =COUNTIF('Master Inventory'!B2:B960,"Received")
  What it means: Count where Status = "Received"
  Example: If 300 parts have arrived, shows 300
  Why: See progress at a glance

PARTS PENDING
  Formula: =COUNTIF('Master Inventory'!B2:B960,"Ordered")
  What it means: Count where Status = "Ordered"
  Example: If 659 still waiting, shows 659
  Why: Know what's still coming

READY TO RETURN
  Formula: =COUNTIF('Master Inventory'!J2:J960,"Return")
  What it means: Count where Return Status = "Return"
  Example: If 5 parts need return, shows 5
  Why: Know what to send back

OVERDUE ITEMS
  Formula: =SUMPRODUCT((I2:I960>30)*(B2:B960="Ordered"))
  What it means: Count of items waiting >30 days AND still Ordered
  Example: 12 parts waiting over a month
  Why: Identifies supplier problems, need escalation

================================================================================
COMMON TASKS
================================================================================

TASK: I just received parts, how do I log them?
  1. Find the part in Master Inventory (use Ctrl+F to search)
  2. In Status column, change to "Received"
  3. In Received Date column, enter today's date
  4. Dashboard auto-updates!

TASK: I want to know how many parts are overdue
  Look at Dashboard sheet → "Overdue Items" metric
  (Shows parts waiting >30 days)

TASK: I want to return something
  1. Find part in Master Inventory
  2. In Return Status column, change to "Return"
  3. Go to Return Tracker sheet
  4. Add row with details (reason, dealership, refund amount)
  5. Track status as it goes back

TASK: I got an invoice, how do I log it?
  Option A (Manual): Type into Invoice Tracker sheet
  Option B (Auto): Use Invoice Parser tool to extract automatically

TASK: I want to see costs per vehicle
  Look at Summary Report sheet
  (Shows total cost, parts count, pending items per vehicle)

TASK: I want to see which dealership is fastest
  Dashboard → Dealership Summary section
  (Shows orders, received, pending per dealership)

================================================================================
TROUBLESHOOTING
================================================================================

Q: Dashboard shows wrong numbers
A: Check that Status values match exactly ("Received" not "RECEIVED")
   Excel is case-sensitive for COUNTIF formulas

Q: Days Waiting column is blank
A: Make sure Order Date column has a date
   Status must = "Ordered" (not "Received")
   Formula only shows waiting time for undelivered items

Q: Can't find a part
A: Use Ctrl+F (Find)
   Sometimes parts are spelled differently
   Try searching for vehicle registration number instead

Q: Invoice doesn't match any order
A: Some parts might have been ordered verbally
   Add a new row to Master Inventory
   Link it to the invoice

Q: Formulas show #REF! error
A: Don't delete rows, hide them instead
   Or ask for help fixing the formulas
